## 🚀 ТРИ ШАГА (5 МИНУТ)

### ШАГ 1: Создайте репозиторий на GitHub (1 минута)
```
1. Откройте https://github.com/new
2. Назовите: MyProjects
3. Выберите: Public
4. Нажмите: Create repository
```

### ШАГ 2: Скопируйте эту команду (1 минута)

**Замените `YOUR_USERNAME` на ваше имя в GitHub!**

```powershell
cd C:\MyProjects && git add . && git commit -m "docs: prepare repository for GitHub publication" && git remote set-url origin https://github.com/YOUR_USERNAME/MyProjects.git && git push -u origin master
```

### ШАГ 3: Запустите команду (3 минуты)
```
1. Откройте PowerShell
2. Вставьте команду (см. выше)
3. Нажмите Enter
4. Готово!
```

## 💻 ЕСЛИ ОШИБКА "Permission denied"

Используйте Personal Access Token:
1. GitHub → Settings → Developer settings → Personal access tokens
2. Generate new token
3. Select: repo, workflow
4. Copy token
5. Используйте token вместо пароля
